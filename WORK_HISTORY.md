# 작업 히스토리

## 프로젝트 개요
포트폴리오 웹사이트 - "100만원을 9억으로 만든 기획자"

## 주요 작업 내용

### 1. 메인 페이지 (index.html)
- 히어로 섹션 구현
- 포트폴리오 보기 버튼에 hover 시 하위 메뉴 표시 기능 추가
- 네비게이션 active 상태 표시 기능

### 2. 프로필 페이지 (journey.html)
- 경력, 학력, 업무 경험 표시
- 스킬 레이더 차트 구현
- SWOT 분석 섹션
- Work 영역: 표 형태를 한 줄 그래프로 변경
  - 전략 기획: 100%
  - 서비스 기획: 100%
  - 회계: 50%
  - 그래프 높이: 22px (기존 32px의 70%)

### 3. 전략 기획 페이지 (work-style.html)
- 1.md: Executive Summary 섹션 추가
- 2.md: 위기 상황과 초기 진단 (Chapter 1) 추가
- 3.md: 전략적 대응 설계 (Chapter 2) 추가
- 4.md: 단계별 투자유치 실행 (Chapter 3) 추가
- 5.md: 성과 및 핵심 역량 (Chapter 4) 추가
- 모든 섹션 반응형 레이아웃 적용
- 다크모드에서 일반모드로 변경
- 모든 박스를 흰색으로 변경

### 4. 향후 계획 페이지 (other-projects.html)
- 카드 배열을 1x4 그리드로 변경
- 다크모드 제거, 일반모드로 변경
- 모든 박스를 흰색으로 변경
- 글꼴 및 색상 통일

### 5. 네비게이션 시스템
- 현재 페이지에 맞는 메뉴에 active 클래스 자동 추가
- active 상태: 글꼴 색상 변경 (#0071e3)
- 포트폴리오 보기 버튼 hover 시 하위 메뉴 표시
- 하위 메뉴 스타일: 버튼과 동일한 색상 및 투명도

### 6. CSS 스타일링
- 반응형 디자인 구현
- 다크모드/일반모드 지원
- 그래프 및 차트 스타일
- 카드 레이아웃 스타일
- 진행 바(Progress Bar) 스타일

## 기술 스택
- HTML5
- CSS3
- JavaScript (Vanilla)
- Chart.js (레이더 차트용)

## 파일 구조
```
포트폴리오/
├── index.html          # 메인 페이지
├── journey.html        # 프로필 페이지
├── work-style.html     # 전략 기획 페이지
├── other-projects.html # 향후 계획 페이지
├── results.html        # 서비스 기획 페이지
├── ai-planner.html     # 진행 중 프로젝트
├── contact.html        # 연락처 페이지
├── assets/
│   ├── css/
│   │   └── style.css   # 메인 스타일시트
│   ├── js/
│   │   └── script.js   # JavaScript 파일
│   └── images/         # 이미지 파일들
└── 1.md ~ 5.md         # 콘텐츠 소스 파일
```
